========================================================================
[멀티모달 평점 예측] 평가 그래프 및 성능 요약표 설명서
========================================================================

1. 실제값 vs 예측값 상관관계 그래프 (상관관계그래프.png)
- X축: 실제 사용자가 부여한 평점 (1~5점)
- Y축: 모델이 예측한 평점
- 설명: 빨간색 회귀선(Trend line)이 45도(y=x)에 가까울수록 예측이 정확함을 의미합니다. 점들이 선에 오밀조밀하게 모여있을수록 분산이 적고 신뢰도 높은 예측을 수행하는 모델입니다.

2. 오차 누적 분포 (오차누적분포.png / Error CDF)
- X축: 절대 오차 (Absolute Error = |실제값 - 예측값|)
- Y축: 누적 비율 (0.0 ~ 1.0)
- 설명: 오차가 작은 구간에서 그래프가 빠르게 1.0(100%)을 향해 올라갈수록 우수한 모델입니다. 예를 들어 X=0.5일 때 Y=0.8이라면, 전체 데이터의 80%가 실제값과 0.5점 이하의 오차를 보인다는 뜻입니다.

3. Model Performance Summary (성능요약.png)
- R-Squared (R²): 모델의 설명력. 1에 가까울수록 모델이 데이터를 잘 설명함.
- MAE (Mean Absolute Error): 평균 절대 오차. 실제값과 예측값 차이의 평균으로, 낮을수록 좋습니다. 이 프로젝트의 핵심 지표입니다.
- RMSE (Root Mean Squared Error): 오차의 제곱에 루트를 씌운 값. 큰 오차(Outlier)에 더 큰 페널티를 부여합니다. 낮을수록 좋습니다.
- MAPE (Mean Absolute Percentage Error): 오차율(%). 예측 오차가 실제값 대비 몇 % 정도인지를 나타냅니다.
- Max Error: 가장 예측이 크게 빗나간 하나의 사례에서의 최대 오차값입니다.
