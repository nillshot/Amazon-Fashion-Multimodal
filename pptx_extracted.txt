프로젝트 수행 배경 
    - 소비 패러다임 변화와 리뷰 데이터의 영향력 증대
이커머스 생태계가 단순 가격 경쟁을 넘어 실제 사용자의 경험이 담긴 리뷰 데이터 중심으로 변모함
따라서, 평점은 소비자의 구매 결정을 좌우하는 핵심 지표이자 정보 비대칭성을 해소하는 일차적 수단이 됨
주관적 만족도가 중요한 의류는 리뷰를 활용해 소비자의 실제 성향을 파악하기에 매우 적합함
    - 패션 산업의 변동성과 기존 예측 방식의 한계
현대 패션 산업은 트렌드 변화가 극심하여 단순 판매량 통계만으로는 소비자의 만족도를 파악하기 어려움
판매량이나 리뷰가 적은 상품의 경우 판매 전략 변경을 하더라도 상품의 흥행 여부를 판단하기 어려움

프로젝트 수행 필요성
    - 비정형 데이터의 정량화 및 자산화
이커머스 시장의 성숙으로 방대하게 축적된 사용자 리뷰를 활용하여 모호한 소비자 반응을 분석하는 자산으로 전환해야 함
    - 평점 예측을 통한 비즈니스 리스크 최소화
전쟁, 유가 상승 등 소비 위축 상황에서 과잉 생산을 억제하고 효율적인 재고 관리 체계를 구축하는 것이 기업 생존을 위해 필요함
신규 상품 출시 전후의 평점을 미리 예측하거나 분석함으로써 타겟 수요에 최적화된 재고 운영 전략을 수립할 수 있음
    - 기술적 한계 돌파를 위한 멀티모달 도입
수치 데이터와 텍스트 리뷰를 별개로 분석하던 기존 방식의 한계를 극복하기 위해 통합 학습 모델이 필요함
데이터 간 유기적 상호작용을 반영하는 멀티모달 아키텍처를 통해 예측 정확도를 높여 실무에서 신뢰할 수 있는 의사결정 도구를 구축해야 함

가설 2. 'helpful_vote'가 높은 리뷰는 데이터 내의 노이즈를 억제하고 신뢰도 높은 정보를 강조하는 가중치 역할을 수행하므로, 모델의 일반화 성능과 예측 정확도를 유의미하게 높일 것	     으로 예상한다.

현재까지의 프로젝트 진행 방향 # 1
    - 아마존 패션 리뷰 데이터를 활용하여 상품의 평점을 예측하는 멀티모달 분석 시스템 구축을 목표로 진행
가설 1(멀티모달의 우수성 증명)과 가설 2(helpful_vote 항목을 활용한 고품질 리뷰 선별)을 바탕으로 연구 설계
가설 검증을 완료하였으며, 이미지 추출기 탐색 과정을 거쳐 융합 매커니즘을 고도화하는 방향으로 모델 아키텍처를 발전 진행
 최종적으로 MoblieNet이 가장 낮은 MAE와 균형잡인 게이트로 우수함이 증명되어 이를 기반으로 연구 진행
    - ‘텍스트, 이미지, 메타 데이터를 융합한 3- way GMU 모델이 단일 모델보다 우수하다’는 가설 1 증명 - 입증
V1 (BaseLine) : 단순 데이터 통합 결과 1.5231로 매우 큰 오차를 보임
V2 (GMU/ Multi-task) : 가중치 조절과 개별 학습을 적용 직후 0.3499로 오차가 급감 (약 77% 감소)
V1과 V2를 통해 단순 나열이 아닌 모달리티 간의 관계를 이해하는 구조가 필수적임을 확인하고 가설 1을 입증
    - ‘helpful_vote 기반 전처리가 노이즈를 억제한다.’는 가설 2 증명 – 기각
원본 데이터(Amazon Reviews'23의 Amazon Fashion)를 모집단으로 하며 각 평점 별 100개 씩 랜덤 추출해 표본 집단 a, b를 구성
표본 집단 a의 helpful_vote에 따라 가중치를 부여한 결과가 표본 집단 b를 가중치 없이 활용한 결과와 큰 차별점을 갖지 못함
가설 2이 기각됨에 따라 helpful_vote를 기반으로 가중치 부여하지 않도록 결정
다만, 저품질 리뷰를 거르기 위해 원본 데이터에서 helpful_vote >= 5를 전처리 조건으로 활용
    - 데이터 파이프라인 및 모달리티 정의
평점이라는 최종 예측 값인 도출하기 위해 각각의 데이터에 적합한 전처리 파이프라인 구축
텍스트는 RoBERTa를 통해 언어적 맥락을 파악하고 이미지는 CNN 기반 백본을 활용해 시각적 피처 추출, 메타 데이터는 MLP을 통해 표현 학습 진행
가격 데이터의 결측치는 중앙값으로 대체하고 결측 여부를 이진 변수로 추가하여 정보 손실 방지 전략 구축

More Than Just Attention: Improving Cross-Modal Attentions with Contrastive Constraints for Image-Text Matching
    - 논문 기본 정보
저자 : Yuxiao Chen, Jianbo Yuan , Long Zhao , Tianlang Chen , Rui Luo , Larry Davis , Dimitris N. Metaxas
발표 기관 [연도] : Rutgers University, Amazon.com Services, Inc [2022년 (v3 기준)]
아카이브 번호 : arXiv:2105.09597v3 [cs.CV]
명시적 주석 없이 대조 제약 조건(CCR, CCS)을 통해 이미지-텍스트 교차 양상 주의(Cross-Modal Attention) 성능을 개선하는 연구
    - 언급된 기존 모델의 문제점 : 답은 맞지만 이유는 모른다.
이미지-텍스트 매칭 모델들은 전체 이미지와 전체 문장이 얼마나 유사한지만을 학습. 그 원인으로 감독 부재와 부정확한 주의(Attention)를 언급
감독 부재 : 개(Dog)라는 단어가 이미지의 어느 영역에 해당하는지 직접 가르쳐주는 데이터가 부족함
부정확한 주의(Attention) : 개(Dog)라는 단어를 처리할 때, 실제 개가 아닌  배경이나 엉뚱한 사물을 보고 관련된 이미지라고 판단하는 오류 발생
    - CCR (Contrastive Content Re-sourcing) 
원리 : 단어와 Attention이 집중된 영역'의 유사도가 Attention이 거의 없는 영역(반전된 영역)과의 유사도보다 월등히 높도록 강제
효과 : 단어와 관련 없는 배경 정보 무시하고 핵심 영역에만 집중하게 함
    - CCS (Contrastive Content Swapping)
원리 : 이미지에서 추출된 특성 정보가 단어와는 잘 어울려야 하지만 관련성 없는 단어와는 적합하지 않도록 함
효과 : 비슷한 위치에 여러 물체 중 대상 단어가 지칭하는 대상을 골라내는 정밀도를 향상시킴
    - 프로젝트 활용 가치 및 도입 시 장점
패션 상품의 평가는 전체적 실루엣보다 넥라인의 형태, 단추의 재질 같은 세밀한 특징에서 나뉨
도입 시 리뷰에 언급된 특정 속성과 실제 영역을 일치시켜 패션 상품의 성공 요인을 더욱 정확하게 파악 가능함
바로 적용 가능한 플러그인 방식이며 추가 데이터가 불필요함
정량적 평가 지표 : 논문이 제안한 지표를 활용한다면 모델이 패션 상품의 디테일을 얼마나 정확히 파악하는지 수치로 증명 가능함

CADMR: Cross-Attention and Disentangled Learning for Multimodal Recommender Systems
    - 논문 기본 정보
저자: Yasser Khalafaoui, Martino Lovisetto, Basarab Matei, Nistor Grozavu
소속 기관(발표 연도) : ALTECA R&D Department, France CY Cergy Paris University, France Sorbonne Paris Nord University, France (2024)
아카이브 정보 (arXiv): arXiv:2412.02295
    - 논문 개요
멀티모달 데이터를 활용해 사용자-아이템 평점 행렬을 복원하고 추천 성능을 향상시키는 모델을 제안
기존 협업 필터링 기반 모델은 희소한 평점 행렬로 인해 성능 저하 문제가 발생. 멀티모달 데이터를 활용하더라도 사용자-아이템 관계와 직접적으로 연결되지 않는 문제가 존재
멀티모달 정보와 사용자-아이템 상호작용을 직접 연결하는 cross-at tention 구조를 통해 보다 정교한 평점 예측을 수행하는 것이 목표
단순 feature 결합이 아닌 관계 기반 attention 수행. multi-head 구조를 통해 다양한 관점에서 정보 학습
    - 모델 구조
Pre-training 단계 : Autoencoder를 활용하여 사용자-아이템 평점 행렬의 잠재 표현 학습 텍스트 및 이미지 특징을 추출하고 disentangled learning을 통해 modality별 특징 분리
Fine-tuning 단계 : cross-attention을 적용하여 평점 행렬과 멀티모달 정보를 결합 강화된 표현을 기반으로 평점 행렬을 재구성
    - Cross-Attention 메커니즘
Q = Rating Matrix (user-item interaction) / K, V = Multimodal Features (text + image)
연산


     - 실험 결과
기존 최신 모델 대비 NDCG 및 Recall 지표에서 성능 향상. 특히 cross-attention을 제거한 경우 성능이 크게 감소하는 것으로 나타나, 모델 성능 향상에 핵심적인 역할임을 확인

--- Slide 11 ---
현재까지 구현한 모델
    - MobileNet 기반 교차 어텐션 멀티모달 회귀 모델
텍스트, 이미지, 정형 데이터의 특징을 개별 추출한 뒤 결합하는 Late Fusion 구조를 기반으로 설계
텍스트 브랜치(RoBERTa) : 자연어 리뷰에 내재된 복합적 맥락과 감정적 의미을 파악하기 위해 RoBERTa를 채택하여 고차원 벡터로 변환 			문맥 임베딩에 최적화되고 비교적 가벼운 모델로 선회하여 초기 계획된 LLaMA-3, Gemma-2와 같은 대규모 언어 모델 대비 효율적
이미지 브랜치(MobileNet-V3 Large) : 상품의 시각적 디테일을 포착하기 위해 도입			경량화 및 효율성이 뛰어난 비전 신경망으로 최종 분류층을 제거한 뒤 특징 추출기를 활용하여 고차원 시각 피처 추출
정형 데이터 브랜치 (MLP) : 가격과 카테고리 등 수치형 및 범주형으로 구성된 상품의 메타 데이터를 처리			메타 데이터를 다층 퍼셉트론 구조에 통과시켜 비선형적 구조와 상관관계 패턴을 학습하도록 구성
융합 레이어 (Cross-Attention Fusion) : 단순 결합의 한계를 극복하기 위해 텍스트와 이미지를 상호 참조하는 교차 어텐션 구조를 채택			이를 통해 수치적인 평점(1점 ~5점)을 정밀하게 예측하는 회귀 모델 구축

모델 선정 이유(MobileNet)
    - 대규모 데이터 처리 효율성 및 과적합 방지
실험 결과, VGG나 ResNet-50 과 같은 무거운 모델은 파라미터가 너무 많아 소규모 벤치마크 셋에서 조기 종료되는 등 불안정한 모습을 보임
수만 건의 전체 데이터 학습 시에도 무거운 모델은 불필요한 연산 자원을 소모하며, 과적합 위험을 높임
MobileNet은 경량화된 구조 덕분에 융합 알고리즘을 수용하면서도 프로젝트 최저 오차(MAE 0.3597)를 도출함
    - 교차 어텐션과의 벨런스
V6 모델에 도입될 대조학습(CCR, CCS)은 연산량이 많음
이미지 백본까지 무거울 경우 전체 아키텍처가 비대해지는 이슈 발생
경량화 구조인 MobileNet은 고도화된 융합 알고리즘을 수용하면서도 최저 오차를 도출함
    - 실무적 가치 창출 및 의사결정 지원이라는 프로젝트 목표에 부합
많은 양의 데이터를 다루는 실무 환경에서 MobileNet은 타 모델 대비 우수한 추론 속도를 보장하여 실무적 환경에 적합

학습 방법 및 하이퍼 파라미터 설정
    - 2-Phase 파인 튜닝
1단계 : 백본 가중치를 동결하여 융합 레이어의 초기 오차 감소시킴
2단계 : 잠금을 해제하여 미세 조정 수행
    - 차등 학습률
모달리티 간의 학습 속도 불균형 해소를 위함
텍스트 인코더(1 * 10-6), 이미지 인코더(1 * 10-5), 융합 헤드(1 * 10-4)에 차등 학습률 부여
    - 다중 작업 손실 함수
최종 결합 예측 외에 텍스트 이미지 단독 회귀 레이어를 선행 통과시켜 개별 오차 최적화함으로써 각 인코더의 전문성 보존
    - 가중 평균제곱오차
데이터가 4 ~5점에 편중된 불균형을 해결하기 위해 빈도가 적은 저평점 오차에 더 큰 페널티 부여
    - 타겟형 모달리티 드롭아웃
텍스트 정보에 대한 과의존을 억제하고 정형 데이터와 이미지 분석에 집중하도록 함
텍스트 데이터를 특정 확률(최대 80%에서 최적화 조정)로 강제 마스킹

모델 구현 과정
    - 데이터 파이프 라인 구축
PyTorch Dataset 및 DataLoader를 활용하여 이미지 크기 조정(224X224) 및 정형 데이터 결측치 처리(0.0 처리 및 Flag 생성) 자동화 로직 구현
    - 아키텍처 진화 (V1 > V6)
단순 결합 베이스라인(V1)에서 시작하여 Multi-task Loss 및 차등 학습률 도입(V2), 텍스트 마스킹 기법(V3)을 적용해 이미지 기여도 점진적 확보
TensorFlow/Keras 환경 호환(V4)을 거쳐 최종적으로 CCR/CCS 결합을 위해 진행 중

Model V1 : Baseline(기본 병렬 결합 모델)
    - 모델 설계
Backbone 구조  : 텍스트 인코더(RoBERTa-base)와 이미지 인코더(EfficientNet-B0)를 독립적인 인코더로 사용하는 3-Way 병렬 구조로 설계
특징 결합 : 각 인코더에서 추출된 고차원 벡터를 단순 결합하여 최종 MLP 레이어에 입력
전처리 전략 : 이미지를 224 X 224 리사이징 및 정규화, 텍스트는 토큰 후 패딩 처리를 통해 입력 포맷 통일
    - 실험 및 평가 방법
최적화 도구 : AdamW 옵티마이저 사용. 평점 예측을 위해 MSE(Mean Squared Error)를 손실 함수로 설정
평가 지표 : 실제 평점과 예측 값 사이 평균 절대 오차를 측정하기 위해 MAE(Mean Absolute Error)를 주요 지표로 활용
학습 전략 : 모든 가중치를 동시에 업데이트 하는 Full Fine-tuning 방식 채택
    - 결과 및 해석
MAE 1.5231로 매우 높게 기록됨과 더불어 이미지 기여도 0%
모델이 복잡한 이미지나 텍스트의 의미를 파악하기 보다 정형 데이터의 상관 관계만 의존하여 학습하는 지름길 학습 현상 발생

Model V2 : Multi-task Learning(동적 가중치 융합 모델)
    - 모델 설계
단순 결합의 한계를 극복하기 위해 GMU 도입하여 모달리티 간의 중요도를 동적으로 결정하는 게이팅 구조로 설계
아키텍처 고도화 : 텍스트와 이미지의 특징이 상호 보완적으로 작용할 수 있도록 어텐션 메커니즘 강화
     - 실험 및 평가 방법
아키텍처 고도화 : 텍스트와 이미지의 특징이 상호 보완적으로 작용할 수 있도록 어텐션 메커니즘을 강화
데이터 불균형 해소 : 5점에 편중된 데이터 특성을 고려하여 Weighted MSE Loss를 도입, 2단계(Freeze-Unfreeze) 전이학습 수행
     - 결과 및 해석
MAE 0.3499. V1 대비 오차율을 약 77% 감소 / 이미지 기여도 5% 미만 기록
정확도는 비약적으로 향상되었으나, 정보 밀도가 높은 텍스트가 모델 전체를 장악하는 텍스트 지배 현상

Model V6 : Intra-Inter Hybrid Attention (최종 고도화 모델)
    - 핵심 기술
Intra-Modality Self-Attention + Inter-Modality Cross-Attention
     - 특징 및 효과
Intra-Attention: 모달리티 내부의 구조적 관계를 먼저 정제하여 정보의 밀도를 극대화.
Inter-Attention: 정제된 텍스트와 이미지 특징 간의 정밀한 교차 참조를 통해 최종 평점 도출.
     - 향후 개선 방향
CCR 및 CCS 대조 학습을 도입할 수 있도록 계획 중에 있음

MobileNet 모델 결과 및 해석
    - 본 프로젝트와 MobileNet의 우수한 적합성
제한된 연산 자원과 데이터 크기에서 MobileNet 계열은 최적의 특징 추출 효율을 입증하며 프로젝트 최종 백본으로 낙점
    - MobileNet 아키텍처 진화(V1~V3)
MobileNet 백본을 뼈대로 삼고 융합 메커니즘을 순차적으로 고도화함에 따라 예측 정확도 스케일이 달라지는 성과를 도출
V1 모델(단순 결합)의 저조한 결과와 달리 V2 모델에서는 GMU 및 Multi-task Learning 손실 함수를 도입하여 예측 오차를 MAE 0.3499로 비약적으로 축소시킴
MobileNet-V3 Large로 백본을 승급시키고 텍스트 데이터를 80% 확률로 강제 마스킹한 V3 모델에서 MAE 0.3461을 유지하며 텍스트 없이도 버티는 모델 강건성 확보
다만 정밀도 향상과는 별개로 이미지 기여도가 5% 미만에 갇혀 있는 병목 지점이 명확하게 관찰됨
    - MobileNet 고도화 (V4/V6) 
V4 메커니즘: Cross-Attention Fusion을 통해 리뷰 키워드와  MobileNet이 추출한 이미지 피처를 상호 매핑
이를 통해 고정밀 오차 (MAE 0.3851)을 달성함과 동시에 이미지 모달리티 기여도를 52%까지 폭발적으로 상승시킴
파라미터가 가벼운 MobileNet의 특성 덕분에 향후 융합이 고려 중인 대조 학습 적용 시 과적합 붕괴 부담이 적음

중간 결론
    - 주요 실험 결과 요약 및 MobileNet의 최종 증명
패션 평점 예측이라는 제한된 데이터의 멀티모달 환경에서 모델의 무조건적인 크기 확장은 긍정적이지 않음을 실험을 통해 확인
파라미터가 비대한 모델은 400개 규모의 데이터셋 환경에서 치명적인 과적합과 일반화 실패를 초래하며 성능 붕괴
반면 MobileNet은 한정된 자원 속에서도 효율적인 피처 추출을 이뤄내며 텍스트 모델(RoBERTa)과의 융합 밸런스를 맞춤
단순 결합(MAE 1.5231)에서 시작해 최종 완성형에 가까운 아키텍처(MAE 0.3597)에 이르기까지 프로젝트 성과 견인
    - 아키텍처 혁신 성과
고질적인 텍스트 편향 일부와 지름길 학습 극복
V4 아키텍처에서 단순 오차 최소화를 넘어 이미지 기여도를 기존 0~5% 대에서 52%까지 끌어올려 멀티모달 정보 시너지를 실증

프로젝트 한계 및 문제점
    - 데이터 불균형에 따른 회귀 편향
전반적인 예측 정밀도는 우수하나 원본 데이터의 태생적 편향 현상이 모델의 약점으로 작용
데이터 분포가 극단적으로 높은 평점에 편향되어 중간 평점값으로 수렴해버리는 '회귀 편향(Regression Bias)' 현상이 백본 선정부터 고도화된 실험에서도 발겸됨
    - 과적합 임계점 및 모달리티 기여도 딜레마
성능-기여도 간의 트레이드오프 현상이 파악됨
결번 처리된 V5에서 MAE 오차는 최고 성능을 냈지만 이미지 기여도는 5% 수준으로 재하락
MAE 오차를 최대한 줄이는 과정에서 가장 정보 밀도가 높은 텍스트 엔진(RoBERTa)에 모델이 다시 의존

향후 계획
    - 분포가 적은 클래스에 더 큰 페널티를 부여하는 등 저평점 구간의 예측 정확도 하락 개선
    - 오차 최소화와 모달리티 융합 균형 간의 최적 타협점 유지. 대조 학습 도입
